# 📖 کتابخانه‌ی داستان‌های یادگیری

> یادگیری مفاهیم فنی با روش **سه‌سطحی**: اول قصه، بعد پشت صحنه، بعد عمق فنی.

## روش سه‌سطحی چیست؟

| سطح | چی یاد می‌گیری؟ | کی بخوانی؟ |
|---|---|---|
| 📗 **سطح ۱ — قصه‌ی مفهومی** | شهودِ موضوع، بدون هیچ کلمه‌ی فنی | اولین برخورد با موضوع |
| 📘 **سطح ۲ — قصه‌ی دقیق** | همان قصه + مکانیزم‌های واقعی، اصطلاحات و دیاگرام‌ها | وقتی شهود را داری و «چطور» می‌خواهی |
| 📕 **سطح ۳ — مرجع فنی** | جزئیات مهندسی، کد، آزمایشگاه عملی، چک‌لیست و سوالات مصاحبه | آمادگی مصاحبه و کار واقعی |

شخصیت ثابت همه‌ی قصه‌ها **سام** است — در هر فصل به دنیای تازه‌ای سر می‌زند. 🧒
