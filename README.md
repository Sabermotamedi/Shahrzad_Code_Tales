# Shahrzad Code Tales

> A story-driven learning repository for computer science, programming, developer fundamentals, and AI concepts, inspired by the timeless storytelling spirit of Shahrzad.

> مخزنی آموزشی و داستان‌محور برای یادگیری علوم کامپیوتر، برنامه‌نویسی، مفاهیم پایه توسعه‌دهندگی و هوش مصنوعی؛ الهام‌گرفته از روایت‌های ماندگار شهرزاد.

---

## English

### What Is This?

`Shahrzad Code Tales` turns technical topics into clear, structured learning stories. The goal is to make difficult concepts easier to understand by combining roadmaps, notes, examples, and progressive explanations.

Instead of memorizing isolated definitions, you follow a learning path that starts with simple intuition and gradually moves toward real technical depth.

### Learning Method

Each major topic is designed around a three-level approach:

| Level | Focus | Best For |
|---|---|---|
| Level 1 | Conceptual story | First contact with a topic |
| Level 2 | Detailed explanation | Understanding how things work |
| Level 3 | Technical reference | Interview prep and real-world depth |

### Contents

- [Stories Library](stories/README.md)
- [Core Computer Science Roadmap](Core_computer_science.MD)
- [AI Developer Roadmap](AI_Roadmap.MD)
- Developer fundamentals, networking, operating systems, databases, architecture, security, DevOps, and AI engineering topics

### Who Is It For?

- Beginner developers who want a clear learning path
- Self-taught programmers building strong foundations
- Students who prefer story-based explanations
- Developers preparing for interviews or deeper technical work

## فارسی

### این مخزن چیست؟

`Shahrzad Code Tales` مفاهیم فنی را به شکل داستان، نقشه‌راه و توضیح مرحله‌به‌مرحله آموزش می‌دهد. هدف این مخزن این است که موضوعات سخت علوم کامپیوتر، برنامه‌نویسی و هوش مصنوعی ساده‌تر، قابل فهم‌تر و ماندگارتر شوند.

در اینجا به جای حفظ کردن تعریف‌های پراکنده، با یک مسیر یادگیری منظم جلو می‌روید؛ از شهود ساده شروع می‌کنید و کم‌کم به عمق فنی و کاربرد واقعی می‌رسید.

### روش یادگیری

هر موضوع مهم با روش سه‌سطحی آموزش داده می‌شود:

| سطح | تمرکز | مناسب برای |
|---|---|---|
| سطح ۱ | قصه‌ی مفهومی | اولین برخورد با موضوع |
| سطح ۲ | توضیح دقیق‌تر | فهمیدن سازوکار واقعی |
| سطح ۳ | مرجع فنی | آمادگی مصاحبه و کار عملی |

### محتوای مخزن

- [نقشه‌راه علوم کامپیوتر](Core_computer_science.MD)

### مناسب چه کسانی است؟

- برنامه‌نویسان مبتدی که دنبال مسیر یادگیری واضح هستند
- افراد خودآموزی که می‌خواهند پایه‌های قوی بسازند
- دانشجویانی که توضیح داستان‌محور را بهتر یاد می‌گیرند
- توسعه‌دهندگانی که برای مصاحبه یا کار جدی‌تر آماده می‌شوند
